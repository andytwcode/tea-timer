# Tea Timer ☕

極簡實用的泡茶倒數計時器，專注於單一功能，提供最佳的泡茶體驗。

## ✨ 功能特色

- 🎯 **極簡設計** - 專注於泡茶倒數，無多餘功能
- ⏱️ **自訂時間** - 輸入分鐘和秒數（5 秒至 10 分鐘）
- 💾 **記住偏好** - 使用 localStorage 記住上次設定的時間
- 📊 **即時顯示** - M:SS 格式清晰顯示剩餘時間
- ✨ **視覺提醒** - 時間到達時文字閃爍 5 次
- 🔄 **重置功能** - 隨時停止並恢復初始時間
- 🏷️ **標題同步** - 瀏覽器分頁標題顯示剩餘時間

## 🚀 快速開始

### 開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev
```

開啟瀏覽器訪問 `http://localhost:5173/`

### 構建

```bash
# 生成生產版本
npm run build

# 預覽構建結果
npm run preview
```

構建產物位於 `dist/` 目錄，可部署到任何靜態網站託管服務。

## 📦 部署

此專案為純靜態網站，可部署到：

- **GitHub Pages** - 使用 GitHub Actions 自動部署
- **Netlify** - 連接 Git 倉庫自動部署
- **Vercel** - 連接 Git 倉庫自動部署

只需將 `dist/` 目錄的內容上傳即可。

## 📖 使用說明

1. **輸入時間** - 分別輸入分鐘數（0-10）和秒數（0-59）
2. **開始倒數** - 點擊「開始」按鈕啟動倒數計時
3. **觀察進度** - 大字體顯示剩餘時間，分頁標題同步更新
4. **時間到達** - 倒數到 0:00 時文字閃爍提醒
5. **重置** - 點擊「重置」按鈕停止並恢復初始時間

### 時間限制

- **最小時長**：5 秒
- **最大時長**：10 分鐘（600 秒）

輸入無效時間會顯示錯誤訊息，且「開始」按鈕將被禁用。

## 📝 注意事項

- 倒數進行時，「開始」按鈕會被禁用
- 切換到其他分頁時，可從分頁標題看到剩餘時間
- 上次設定的時間會自動儲存，重新開啟時自動載入
- 僅提供視覺提醒，無音效或震動
