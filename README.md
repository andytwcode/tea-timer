# Tea Timer ☕

極簡實用的泡茶倒數計時器，專注於茶藝愛好者的需求，提供最佳的泡茶體驗。

## ✨ 功能特色

### 核心計時功能
- 🎯 **極簡設計** - 專注於泡茶倒數，無多餘功能
- ⏱️ **自訂時間** - 輸入分鐘和秒數（5 秒至 10 分鐘）
- ⏸️ **暫停/繼續** - 計時中可隨時暫停和恢復，保留剩餘時間
- 🔄 **重置功能** - 隨時停止並恢復初始時間
- 📊 **即時顯示** - M:SS 格式清晰顯示剩餘時間
- 🏷️ **標題同步** - 瀏覽器分頁標題顯示剩餘時間和泡數

### 連續沖泡模式
- 🍵 **多泡管理** - 專為功夫茶設計，支援連續多泡沖泡
- ⏰ **增量時間** - 每泡自動增加指定時間（如 30 秒）
- 📈 **泡數顯示** - 清楚顯示當前第幾泡
- ⚙️ **可摺疊設定** - 增量設定區域可收起，保持介面簡潔

### PWA & 通知
- 📱 **可安裝應用** - 支援 PWA，可安裝到手機桌面
- 🔔 **完成通知** - 倒數結束時發送系統通知（支援多平台）
- 🔊 **聲音震動** - Android 裝置上的通知支援聲音和震動
- 💾 **離線使用** - Service Worker 支援，可離線運行

### 資料持久化
- 💾 **記住偏好** - 使用 localStorage 記住上次設定
- 🔄 **自動載入** - 重新開啟時自動載入上次的時間設定
- 📝 **智慧摺疊** - 記住增量設定的展開/收起狀態

## 🚀 快速開始

### 開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev
```

開啟瀏覽器訪問 `http://localhost:5173/`

### 構建

```bash
# 生成生產版本
npm run build

# 預覽構建結果
npm run preview
```

構建產物位於 `dist/` 目錄，可部署到任何靜態網站託管服務。

此專案為純靜態網站，可部署到：

- **GitHub Pages** - 使用 GitHub Actions 自動部署
- **Netlify** - 連接 Git 倉庫自動部署
- **Vercel** - 連接 Git 倉庫自動部署

## 🛠️ 技術棧

- **Vue 3** - 響應式前端框架（Composition API）
- **Vite** - 極速構建工具
- **Tailwind CSS** - Utility-first CSS 框架
- **PWA** - Service Worker 支援離線和通知功能

### 程式架構
- **Composables 模式** - 使用 Vue 3 Composables 實現關注點分離
  - `usePWA` - PWA Service Worker 註冊
  - `useLocalStorage` - 持久化存儲管理
  - `useNotification` - 通知系統（支援跨平台）
  - `useTimer` - 計時器核心邏輯
  - `useMultiSteep` - 連續沖泡邏輯

## 📖 使用說明

### 基本操作
1. **輸入時間** - 輸入第一泡的分鐘數（0-10）和秒數（0-59）
2. **開始倒數** - 點擊「▶ 開始」按鈕啟動倒數計時
3. **暫停/繼續** - 計時中點擊「⏸ 暫停」可暫停，再點擊「▶ 繼續」恢復
4. **觀察進度** - 大字體顯示剩餘時間，分頁標題同步更新
5. **時間到達** - 倒數到 0:00 時文字閃爍並發送通知
6. **重置** - 點擊「🔄 重置」按鈕停止並恢復初始時間

### 連續沖泡模式
1. **啟用連續沖泡** - 勾選「啟用連續沖泡」選項
2. **設定增量時間** - 點擊「⚙️ 設定」展開，輸入每泡增加的時間（如 0 分 30 秒）
3. **開始第一泡** - 點擊「▶ 開始第 1 泡」
4. **完成後繼續** - 第一泡完成後，點擊「▶ 開始第 2 泡」繼續下一泡
5. **結束沖泡** - 所有泡數完成後，點擊「🛑 結束沖泡」重置泡數

### 通知設定
1. **首次使用** - 開啟網站時會請求通知權限
2. **允許通知** - 點擊「允許」以啟用完成提醒
3. **跨平台支援** - 桌面和 Android 裝置都能收到通知

### 時間限制

- **最小時長**：5 秒
- **最大時長**：10 分鐘（600 秒）

輸入無效時間會顯示錯誤訊息，且「開始」按鈕將被禁用。

## 📝 注意事項

### 計時功能
- 計時中可隨時暫停，保留剩餘時間
- 暫停時輸入欄位仍然鎖定，需重置才能修改
- 切換到其他分頁時，可從分頁標題看到剩餘時間和暫停狀態

### 連續沖泡
- 每泡的時間 = 第一泡時間 + (泡數-1) × 增量時間
- 增量時間可設為 0，此時所有泡數時間相同
- 取消勾選連續沖泡時，增量設定會自動收起

### 資料儲存
- 所有設定自動儲存到 localStorage
- 包含時間設定、增量設定、連續模式開關、摺疊狀態
- 重新開啟網站時自動載入上次設定

### 通知功能
- 桌面裝置：使用瀏覽器原生通知 API
- Android 裝置：透過 Service Worker 發送通知（支援聲音震動）
- iOS 裝置：受限於系統限制，可能無法使用通知
- 需要授予通知權限才能接收提醒

### PWA 安裝
- 支援 Android Chrome 和桌面 Chrome 安裝
- 安裝後可從桌面啟動，體驗更接近原生 App
- 離線也能使用（已安裝的情況下）
