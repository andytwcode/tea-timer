# Proposal: 可摺疊的增量設定區域

## Why

手機螢幕空間有限，連續沖泡的「每泡增加時間」設定區域佔用較多垂直空間。由於使用者通常只在初次設定時調整增量（之後很少更改），預設展開的設定介面造成不必要的空間浪費。提供可摺疊的設定區域能夠在保持功能完整性的同時，改善手機使用體驗。

## What Changes

- 增量設定區域新增展開/收起功能
- 預設狀態為收起（collapsed），節省畫面空間
- 收起時顯示「⚙️ 設定增量 (目前：X分Y秒)」按鈕，讓使用者了解目前設定值
- 展開時顯示「▲ 收起」按鈕，可調整增量分鐘和秒數
- 展開/收起狀態記憶到 localStorage，保持使用者偏好
- 取消勾選「啟用連續沖泡」時，自動收起設定區（如果展開的話）

## Capabilities

### New Capabilities

無。此變更為現有功能的 UI 優化，不引入新的核心能力。

### Modified Capabilities

- `multi-steep-control`: 增量輸入區域改為可摺疊設計，新增展開/收起按鈕與狀態管理
- `storage-persistence`: 新增儲存與讀取增量設定區域的展開/收起狀態

## Impact

**影響的檔案**：
- `src/App.vue`: 增量輸入區域的 UI 結構與互動邏輯

**localStorage 變更**：
- 新增 `teaTimerShowIncrementSettings` 鍵（boolean），記錄展開/收起狀態

**使用者體驗改善**：
- 手機版畫面更簡潔
- 減少滾動需求
- 保留完整功能可用性
